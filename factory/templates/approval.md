# Approval Record

No external repository write is allowed until an explicit approval entry exists.

`target_repo` and `write_scope` are not approval by themselves. The run creator must provide explicit approval text.

Required fields for a real approval entry:

- marker: literal approval marker generated by the factory after `--approval-text`
- timestamp: UTC ISO-8601
- target_repo: absolute path or repo URL
- write_scope: exact paths/globs
- approved_by: `yalru`
- approval_text: user request or scoped approval
