# UI/UX QA Checklist

- [ ] `DESIGN.md` exists and controls tokens.
- [ ] The UI is a real product screen.
- [ ] No requirements chips or developer meta UI are visible.
- [ ] 375px layout has no clipping, overlap, or horizontal scroll.
- [ ] 768px layout remains coherent.
- [ ] 1280px layout remains coherent.
- [ ] BottomCTA, if present, is fixed to bottom and safe-area aware.
- [ ] Default/hover/focus/disabled/loading/error/empty states are covered as applicable.
- [ ] Buttons have accessible names.
- [ ] Keyboard focus order is usable.
- [ ] Contrast is acceptable.
- [ ] Screenshot evidence is stored under `evidence/`.
